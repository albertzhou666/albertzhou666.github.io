[What is API Versioning in REST? (restfulapi.net)](https://restfulapi.net/versioning/)

Examples

```http
# uri examples
http://www.example.com/api/v1/products
http://api.example.com/v1/products
http://www.example.com/api/products?version=1

# http header
Accept-version: v1

# extension header
Accept: version=1.0
```
